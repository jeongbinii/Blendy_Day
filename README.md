# Blendy_Day

> **"당신의 대화가 일기가 되는 마법"**
> 감정 기록 및 공감 매칭 플랫폼

내 마음을 들여다보고 글을 쓰는 것이 막막한 분들을 위해 기획되었습니다.
Blendy Day는 친근한 AI와의 채팅을 통해 자연스럽게 감정을 이끌어내고, 완성된 일기를 바탕으로 비슷한 하루를 보낸 타인과 따뜻한 위로를 나눌 수 있는 새로운 형태의 **소셜 다이어리**입니다.

---

## 🎯 핵심 유저 플로우 (How It Works)

### 1. AI와 편안하게 대화하기 💬
- "오늘 하루 어땠어?"로 시작하는 AI와의 채팅
- 감정 표현이 서툰 사용자에게 AI가 적절한 질문을 던져 숨겨진 감정과 에피소드를 자연스럽게 끌어냄

### 2. 초안 생성 및 사용자 맞춤 편집 ✍️
- 대화가 종료되면 AI가 내용을 깔끔하게 정리하여 **'오늘의 일기 초안'** 작성
- 사용자는 AI의 문장을 자신의 말투로 수정 가능
- 최종 발행 권한은 오직 사용자에게 있음

### 3. 감정 스티커 부착 🏷️
- 최종 업로드 전, 오늘 하루를 요약하는 **감정 스티커** 부착
- 행복 😊 / 그럭저럭 😑 / 슬픔 😢 세 가지 감정으로 분류

---

## ✨ 주요 기능 (Key Features)

### 📊 AI 감정 패턴 분석 리포트
- 긍정/부정 일기 데이터를 AI가 심층 분석
- "주로 어떤 요일에, 어떤 상황에서" 감정을 느끼는지 **메타인지(Self-awareness)** 를 돕는 통계 제공

### 🤝 해시태그 기반 익명 랜덤 매칭
- 완성된 일기에서 핵심 단어(해시태그)를 AI가 자동 추출
  (예: `#퇴사고민` `#비오는날` `#소소한행복`)
- 같은 해시태그를 가진 사용자들끼리 랜덤 매칭되어 익명으로 공감과 위로 교환

### 🎭 다중 페르소나 AI 챗봇
- 사용자의 취향에 맞춰 AI의 성격 선택 가능
- T성향의 냉철한 조언자, F성향의 따뜻한 공감러 등 다양한 캐릭터 모델 지원

## 💼 비즈니스 모델 (Business Model)

| 카테고리 | 수익화 전략 | 상세 내용 |
|---|---|---|
| 페르소나 스토어 | 맞춤형 AI 캐릭터 판매 | 기본 제공 AI 외, 특별한 대화 스타일(MBTI, 유명인 컨셉, 직업군 등)을 가진 캐릭터를 유료 구매 또는 구독 |
| 소셜 매칭 | 랜덤 매칭 티켓 판매 | 기본 제공 일일 랜덤 매칭 횟수 초과 시 매칭권 추가 결제 |
| 프리미엄 리포트 | 심층 심리 분석 구독 | 주간/월간 단위의 정밀한 감정 분석 리포트 제공 (SaaS 형태) |

## 📁 프로젝트 구조 (Project Structure)

```
src/
├── components/
│   └── Layout.jsx          # 공통 레이아웃 (네비게이션 포함)
├── constants/
│   └── emotions.js         # 감정 스티커 상수 (행복/그럭저럭/슬픔)
├── pages/
│   ├── Home.jsx            # 랜딩 페이지
│   ├── Login.jsx           # 로그인 페이지
│   ├── Chat.jsx            # AI 채팅 페이지
│   ├── DiaryEdit.jsx       # 일기 편집 페이지
│   ├── DiaryView.jsx       # 일기 조회 페이지
│   ├── Dashboard.jsx       # 대시보드 (일기 목록)
│   ├── Report.jsx          # 감정 분석 리포트
│   └── Match.jsx           # 익명 공감 매칭
└── App.jsx                 # 라우팅 설정
```

### 라우팅 구조

| 경로 | 페이지 | 설명 |
|---|---|---|
| `/` | Home | 서비스 소개 랜딩 페이지 |
| `/login` | Login | 로그인 / 회원가입 |
| `/chat` | Chat | AI와 대화하여 일기 작성 |
| `/diary/edit` | DiaryEdit | AI가 생성한 일기 초안 편집 |
| `/diary/view` | DiaryView | 완성된 일기 열람 |
| `/dashboard` | Dashboard | 내 일기 목록 및 감정 통계 |
| `/report` | Report | AI 감정 패턴 분석 리포트 |
| `/match` | Match | 해시태그 기반 익명 매칭 |

## 🎨 디자인 가이드

**컨셉: 웜 미니멀리즘 (Warm Minimalism)**
과한 그래픽을 배제하고 여백의 미를 살리며, 스크롤 시 종이가 넘어가는 듯한 부드러운 트랜지션 애니메이션 적용.

| 역할 | 컬러 | 코드 | 설명 |
|---|---|---|---|
| 메인 컬러 | Muted Blue/Gray | `#7895B2` | 차갑지 않은 톤다운 블루그레이 — 차분함, 신뢰감 |
| 배경 컬러 | Oatmeal (웜 화이트) | `#F9F8F6` | 눈이 부시지 않은 따뜻한 배경색 |
| 포인트 컬러 | 파스텔톤 | 머스타드 옐로우, 세이지 그린 등 | 채도를 낮춰 감정 스티커에 사용, 튀지 않게 블렌딩 |

### 감정 스티커 컬러

| 감정 | 이모지 | 배경색 | 설명 |
|---|---|---|---|
| 행복 | 😊 | `#fdf6e0` / `#f0c05a` | 따뜻한 옐로우 계열 |
| 그럭저럭 | 😑 | `#f0f4f8` / `#7895B2` | 차분한 블루그레이 계열 |
| 슬픔 | 😢 | `#edf1f9` / `#a0b4d0` | 은은한 블루 계열 |
